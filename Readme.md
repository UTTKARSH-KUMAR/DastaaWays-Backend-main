DastaaWays is an intelligent AI meeting assistant that listens, understands, and documents project requirements in real-time.  
Inspired by the Hindi word **लिपिकार (scribe)**, DastaaWays transforms client conversations into structured **Functional Requirement Documents (FRD)** or **Software Requirement Specifications (SRS)** automatically — eliminating manual note-taking.

---

## 🚀 Overview

During requirement meetings, important details are often missed, forgotten, or poorly documented.  
DastaaWays solves this by acting as a real-time AI scribe that records conversations, extracts requirements, asks clarifying questions, and generates professional documentation instantly.

This makes it ideal for:

- Software development teams
- Product managers
- Business analysts
- Freelancers & agencies
- Startup founders

---

## ✨ Key Features

### 🎙 Real-Time Meeting Transcription

- Converts speech to text using AI-powered speech recognition
- Works during live meetings
- Supports long conversations

### 🤖 Smart Requirement Extraction

- Detects functional requirements
- Detects non-functional requirements
- Detects business rules automatically

### ❓ Clarifying Question Engine

- Finds missing details in requirements
- Asks follow-up questions automatically
- Ensures complete documentation

### 📂 Automatic Document Generation

- Generates structured FRD / SRS
- Export formats:
  - PDF
  - DOCX
  - Markdown
- Clean and professional formatting

### 🔍 Requirement Categorization

- Must-have requirements
- Nice-to-have features

- Risks / assumptions

### 📊 Integration-Ready

- Jira

- Google Docs
- API support for custom tools
